# iamhalje.github.io

Personal landing page for GitHub Pages.

## What to edit first

- Update your email in `index.html` (`mailto:your.email@example.com`)
- Update/verify social links (`github.com/iamhalje`, `linkedin.com/in/halje`)
- Update Telegram link (`t.me/halje`)
- Fill the “Projects” section with real repositories

## Publishing

In GitHub repository settings:

- Settings → Pages
- Source: Deploy from a branch
- Branch: `main` (or the default branch) / root

Then the site will be available at `https://iamhalje.github.io/`.
